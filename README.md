# ControleDeContas
#SpringBoot
#Back-end

Projeto criado para controle de contas bancárias;

Funcionalidades : 
  Criar conta, 
  Consultar Conta, 
  Inserir Crédito, 
  Transferência Bancária, 
  Consulta de Saldo, 
  Bloquear e Desbloquear conta,
  Documentação automática com swagger.
